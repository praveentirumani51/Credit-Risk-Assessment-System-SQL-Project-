# Credit-Risk-Assessment-System-SQL-Project-
A complete end-to-end risk scoring and segmentation engine built using SQL and core lending principles. This project simulates how banks evaluate borrowers using real underwriting logic: EMI, DTI, LTV, behaviour flags, CIBIL, DPD, and inquiries.
Project Highlights

Built a risk scoring system integrating EMI, DTI, LTV, utilization, inquiries, and DPD.
Segmented borrowers into Prime, Good, Medium Risk, High Risk groups.
Identified red flags such as 80%+ utilization, 30+ DPD, 5+ inquiries.
Generated portfolio insights on CIBIL, DTI trend, DPD increases, and utilization shifts.
Created automated underwriting logic to assist loan approval/decline workflows.
